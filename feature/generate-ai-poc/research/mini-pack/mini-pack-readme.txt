Mini-pack README

ขอบคุณที่สนใจทดลองใช้งาน mini-pack ของเรา (AiStidio mini‑pack)

ไฟล์ในโฟลเดอร์นี้:
- mini-pack-readme.txt (ไฟล์นี้)
- template-link.txt (ลิงก์ไปยัง Canva template หรือคำอธิบายการใช้งาน)
- preview-instructions.txt (คำแนะนำการสร้างภาพตัวอย่าง / mockup)
- subtitle-example.srt (ตัวอย่างไฟล์ subtitle)
- sample-audio-info.txt (คำอธิบายสำหรับไฟล์เสียงตัวอย่าง หากต้องการอัปโหลด)

การใช้งานเบื้องต้น
1. หากต้องการแจกจริง: อัปโหลดไฟล์ template (.zip/.png/.pdf) ขึ้น Google Drive/Dropbox แล้วตั้งค่าการแชร์เป็น "Anyone with the link" แล้ววางลิงก์ลงในไฟล์ template-link.txt
2. หากต้องการให้ระบบส่งอีเมลอัตโนมัติ: ใช้ Apps Script ที่เตรียมไว้ก่อนหน้านี้ (ปรับ download link)
3. แนะนำให้เตรียม preview image ขนาด 1080x1920 (vertical), thumbnail 1200x628, และ grid preview 1024x1024 เพื่อใช้เป็นภาพโชว์ใน listing

ติดต่อ
- หากต้องการให้ผมอัปโหลดไฟล์จริงให้ กรุณาอัปโหลดไฟล์ไปให้ผม (หรือให้ลิงก์) แล้วผมจะช่วยวางลงใน repository หรือเตรียม Apps Script ให้เรียบร้อย
